# 五子棋游戏 (Gomoku)

一个精美的五子棋网页游戏，支持双人对战和人机对战模式。

## 🎮 功能特性

### 游戏模式
- **👥 双人对战** - 两人轮流落子
- **🤖 人机对战** - 与AI对手博弈

### 游戏功能
- ✅ 15×15 标准棋盘
- ✅ 五子连珠判断（横/竖/斜）
- ✅ 悔棋功能
- ✅ 比分记录（本地存储）
- ✅ 最后一步高亮显示
- ✅ AI思考动画指示

### 音效系统
- 🔊 落子音效
- 🎉 胜利音效
- 🤝 平局音效
- ↩️ 悔棋音效
- 🔄 重开音效
- 🔇 音效开关

### AI特性
- 智能棋型评估
- 进攻与防守策略
- 多方向威胁检测

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html`
2. 选择游戏模式（双人对战/人机对战）
3. 点击棋盘交叉点落子
4. 先连成五子者获胜

## 🎯 操作说明

| 操作 | 说明 |
|------|------|
| 点击棋盘 | 落子 |
| 悔棋 | 撤销上一步（AI模式撤销两步） |
| 重新开始 | 重置棋盘 |
| 音效按钮 | 开关音效 |

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式与动画
- **JavaScript (ES6)** - 游戏逻辑
- **Web Audio API** - 音效生成
