# 🎮 五子棋游戏

一个纯前端的五子棋小游戏。

## 功能特性

- ✅ 15×15 标准棋盘
- ✅ 黑白双方轮流落子
- ✅ 自动判断胜负（五子连珠）
- ✅ 平局检测
- ✅ 悔棋功能
- ✅ 比分记录
- ✅ 最后一步高亮标记

## 使用方法

双击 `index.html` 在浏览器中打开即可开始游戏。

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
